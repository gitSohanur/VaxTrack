# VaxTrack — Vaccine Tracking System

A Java OOP university project built with JavaFX, MySQL, and JDBC.

## Tech Stack
- Java 17
- JavaFX 21
- MySQL 8
- JDBC
- Maven

## Setup
1. Clone the repo
2. Create MySQL DB using the script in `/docs/schema.sql`
3. Update DB credentials in `DatabaseConfig.java`
4. Run via IntelliJ or `mvn javafx:run`
